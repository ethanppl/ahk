# AutoHotKey

## Remap

- `CapsLock` key is the same as `Ctrl` key
- Use `Shift+Capslock` to toggle `Capslock`

## Hot Keys

| Hot Key | Meaning                 |
| ------- | ----------------------- |
| `z+d`   | Scroll down a few lines |
| `z+u`   | Scroll up a few lines   |
| `z+j`   | Scroll down 1 line      |
| `z+k`   | Scroll up 1 line        |
